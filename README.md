# Gmail Automation Script (Python + Selenium)

This project uses Selenium to automatically open a Chrome browser window and navigate to the Gmail homepage.

---

## Installation

### 1. Install Python  
Download & install Python from:

https://www.python.org/downloads/

✔ Make sure to enable **Add Python to PATH** during installation

---

### 2. Install Required Dependencies

Run the commands below:

```sh
pip install selenium
pip install webdriver-manager
````

---

## How to Run

1. Save the provided Python script as:

```
code.py
```

2. Run the script:

```sh
python code.py
```

A Chrome browser window will open and navigate to:

```
https://mail.google.com/
```
